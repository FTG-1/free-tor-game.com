C.1 FR1 Channel models

38.1513GPPNRRelease 17TSUser Equipment (UE) Multiple Input Multiple Output (MIMO) Over-the-Air (OTA) performance requirements

Tools: ARFCN - Frequency Conversion for 5G NR/LTE/UMTS/GSM

The following channel models are required for FR1 MIMO OTA measurement.

The generic models are Table C.1-1 FR1 UMi CDL-C and Table C.1-2 FR1 UMa CDL-C, which do not include base station antenna filtering. UMi CDL-C and UMa CDL-C are selected to define 2×2 and 4×4 MIMO OTA requirements, respectively.

Therefore, in addition, the BS beam filtering effect defined in Annex C.2 also apply when emulating the channel models.
